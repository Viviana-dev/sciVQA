# SciVQA  
## Zero-shot setting 
- BLIP2 - 2.7B
- LLAVA - 13B , LLAVA - 7B
- OpenFlamingo v2 - 7B
- mPLUG-owl-7B (Flamingo)
- GPT3

## Transformation of graphs/charts (chart data extraction)
- OGR (Optical Graph Recognotion) / OCR (Optical character recognition) eg. ChartOCR model
- DePLot
- Matcha
- Pix2Struct
- StructChart

## Metrics
- BERTscore
- ROUGE-L
- ROUGE-1
